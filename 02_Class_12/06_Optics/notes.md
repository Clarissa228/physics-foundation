# Optics

## Overview
Light obeys both ray laws (geometric optics) and wave principles (wave optics).

## Key Concepts

### 1. Law of Reflection
θ_i = θ_r (angle of incidence = angle of reflection)

### 2. Snell's Law (Refraction)
$$n_1\sin\theta_1 = n_2\sin\theta_2$$

where n is refractive index.

**Critical angle**: $\sin\theta_c = n_2/n_1$ (for total internal reflection)

### 3. Thin Lens Formula
$$\frac{1}{f} = \frac{1}{v} + \frac{1}{u}$$

where:
- f = focal length
- u = object distance  
- v = image distance

**Magnification**: $m = -v/u$

### 4. Lens Types
- **Converging (convex)**: f > 0, brings rays together
- **Diverging (concave)**: f < 0, spreads rays apart

**Lensmaker's equation**: $\frac{1}{f} = (\mu-1)(\frac{1}{R_1} - \frac{1}{R_2})$

### 5. Young's Double Slit
Path difference: Δ = d sin θ ≈ dy/D

**Bright fringes**: Δ = mλ (m = 0,1,2,...)
**Dark fringes**: Δ = (m+1/2)λ

Fringe spacing: Δy = λD/d

### 6. Diffraction - Single Slit
First minimum: a sin θ = λ (a = slit width)

Intensity: $I = I_0 (\frac{\sin\beta}{\beta})^2$ where $\beta = \frac{\pi a\sin\theta}{\lambda}$

### 7. Diffraction Grating
$$d\sin\theta = m\lambda$$

where d = grating spacing, m = order

### 8. Thin Film Interference
Path difference: 2μt cos r (μ = refractive index, t = thickness)

**Bright**: 2μt cos r = mλ (constructive)
**Dark**: 2μt cos r = (m+1/2)λ (destructive)

### 9. Polarization
**Malus's Law**: $I = I_0\cos^2\theta$

where θ = angle between polarizers.

### 10. Optical Instruments
**Microscope magnification**: M = m_obj × m_eye
**Telescope angular magnification**: M = -f_obj/f_eye
